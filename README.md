# 📚 StudyFlow

A modern and minimal productivity web application built with **React** to help students stay organized and focused.

StudyFlow combines a **Task Manager**, **Notes App**, and **Pomodoro Timer** into one clean dashboard with persistent data using **Local Storage**.

---

## 🚀 Features

### 📋 Task Manager
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Completed tasks are shown with a strikethrough
- Data persists using Local Storage

---

### 📝 Notes
- Create notes with title and description
- Delete notes
- Edit existing notes
- Notes are automatically saved in Local Storage

---

### 🍅 Pomodoro Timer
- 30-minute focus timer
- Start, Pause and Reset controls
- Countdown timer (MM:SS format)
- Automatically stops when timer reaches zero
- Tracks completed focus sessions

---

### 📊 Dashboard
- Total notes count
- Pending task cound
- Pomodoro 
- Clean and responsive UI

---
